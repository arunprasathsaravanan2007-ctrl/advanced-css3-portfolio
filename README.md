# Advanced CSS3 Responsive Portfolio

A modern, responsive portfolio website built using HTML5, CSS3, and JavaScript. This project demonstrates advanced CSS3 concepts, responsive web design principles, CSS Grid, Flexbox, and a mobile-first development approach.

## 🚀 Features

- Responsive Design for Mobile, Tablet, and Desktop
- CSS Grid Layout
- Flexbox Layout
- Mobile-First Architecture
- Dark/Light Theme Toggle
- Smooth Scrolling Navigation
- Interactive Hover Effects
- Modern UI Components
- Contact Form
- Sticky Navigation Bar
- Reusable Card Components

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📱 Responsive Design

The website is designed using a mobile-first approach and adapts seamlessly across:

- Mobile Devices
- Tablets
- Laptops
- Desktop Screens

## 🎨 CSS Concepts Implemented

### CSS Grid

Used for:

- Hero Section Layout
- About Section Layout
- Skills Section
- Projects Section

### Flexbox

Used for:

- Navigation Bar
- Form Layouts
- Content Alignment
- Card Components

### CSS Variables

```css
:root{
    --primary:#6c63ff;
    --secondary:#2d2d39;
    --bg:#0f172a;
    --card:#1e293b;
    --text:#ffffff;
}
```

### Media Queries

Used to create responsive layouts for different screen sizes.

### Theme Switching

Supports:

- Dark Theme
- Light Theme
